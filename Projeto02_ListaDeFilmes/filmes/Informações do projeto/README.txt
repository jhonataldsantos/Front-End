== Ferramentas utilizadas ==
Back-End: IntelliJ
Front-End: VSCode
Banco de Dados: PostgreSQL

== Explicando o Back - End ==
Escolhi realizar no IntelliJ pela praticidade da IDE com o Spring Boot. O código está comentado e está realizando os métodos HTTP GET e POST, necessárias para o projeto. (GET para listar os filmes presentes no banco de dados, e POST para incluir novos filmes na tabela)

== Explicando o Front - End ==
Escolhi realizar no VSCode por familiaridade com a IDE. Optei por realizar um HTML recebendo as informações pelo script.js, e realizando o design pelo style.css.
